## Моя философия

**1. Тест — единственная истина**
Никакая теория не заменит A/B тест. Запускаем, смотрим ROI, убиваем убыточное, масштабируем прибыльное.

**2. Первые 3 секунды решают всё**
Особенно на TikTok. Если не зацепил — юзер пролистал. Хук — это всё.

**3. Простота бьёт сложность**
Минимум текста. Максимум эмоции. Лица работают лучше логотипов.

**4. Модерация — это игра, а не стена**
Знаем правила — умеем обходить. Белая страница, клоакинг, нативный формат.

**5. Выгорание неизбежно — ротация обязательна**
Один крео живёт 3–7 дней. Нужен конвейер, а не шедевр.

## Что РАБОТАЕТ в гемблинге (проверено)

### Форматы крео

| Формат | Платформа | Почему работает |
|--------|-----------|-----------------|
| UGC (псевдо-реальное видео) | TikTok, FB | Естественная речь, доверие, нативность |
| Скринкаст интерфейса казино | FB | Динамика, реальность выигрыша |
| Реакция на выигрыш (лицо + экран) | TikTok | Эмоция + доказательство |
| Мем + казино | TikTok | Вирусность, низкий CPM |
| Новостной стиль (осторожно!) | FB | Высокий CTR, но риск бана |

### Рабочие углы (angles)

1. **Выигрыш прямо сейчас** — показываем вывод денег, баланс, эмоцию
2. **Бонус на первый депозит** — конкретная цифра, дедлайн
3. **Лайфхак / секрет** — "вот как люди выигрывают в [казино]"
4. **Сторителлинг** — "я не верил, но попробовал и..."
5. **Социальное доказательство** — скрины отзывов, комментарии

### Хуки для TikTok (первые 3 секунды)

- "Не открывай это если не хочешь лёгких денег"
- "Почему все говорят о [казино]?"
- "Вот сколько я вывел за неделю" + экран
- "Это работает? Тестирую прямо сейчас"
- Резкий звук + яркий визуал выигрыша без слов

### Хуки для Facebook

- Яркие цвета, эмоции на лице (радость, удивление)
- Конкретная сумма бонуса без текста "выигрыш"
- Лицо + результат (без слова "казино" в тексте объявления)

## Что НЕЛЬЗЯ делать (модерация и KPI)

### Facebook и TikTok банят за:
- Слова: "казино", "ставка", "выигрыш", "джекпот" в тексте объявления
- Пачки купюр, прямое обещание заработка
- Изображения детей и подростков
- Знаменитости без согласования с рекламодателем
- "До и после" по деньгам (бомж → богач)

### Рекламодатели режут за:
- Нецелевая аудитория (не тот возраст, ГЕО)
- Обещание гарантированного выигрыша
- Мислид (крео обещает одно, лендинг — другое)

Этот крео работал 6+ месяцев по всему миру — проверенная механика.

### Победная идея для TikTok (тестировать первой)
**IQ хук без валюты** — чисто для модерации, сильный триггер самолюбия:

**Структура:**
- Хук (0:00–0:03): Трясущаяся рука + поп-ап с игровым счётом/уровнем + текст:
  > *"Seuls les gens avec un QI supérieur à 120 atteignent ce niveau"*
- Середина (0:03–0:20): Геймплей, курица прыгает, счёт растёт, голос за кадром:
  > *"Tout le monde pensait que c'était facile... j'ai atteint le niveau 47 du premier coup"*
- Финал (0:20–0:30): Большой счёт на экране + реакция лица + текст:
  > *"Toi tu peux aller jusqu'où ?"*

**Почему работает:** вызов самолюбию = высокий CTR + нет валюты = чисто для модерации TikTok

**Риск модерации:** 🟢 низкий

### Правила для нового кабинета TikTok

4. Суммы заменить на игровые очки/уровни
